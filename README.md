# Resurgence-Map-Calculator
Resurgence Map Calculator (RMC) is a tool for Pavlov Shack Resurgence I made to find where airdrops are on on the map and calculate the distance between your location and the nearest points of interest

This tool is not perfect so if the points or chords are ever so slightly off thats why

A more in depth read me might be added at some point but for now everything is pretty self explanitory.
